# 🎓 **College Enquiry Chatbot**





#### 📌 Description

###### This is a Generative AI-based chatbot that answers college-related queries like courses, fees, hostel, and admission.



#### 🚀 Features

###### \- AI-powered responses

###### \- Simple GUI using Streamlit 

###### \- Fast and interactive



#### 🛠️ Tech Stack

###### \- Python

###### \- Streamlit

###### \- Open AI API

#### 

#### ▶️ How to Run

###### 1\. Install dependencies:

###### &#x20;  pip install streamlit open-ai

###### 

###### 2\. Run the app:

###### &#x20;  streamlit run app.py
